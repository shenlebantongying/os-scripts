## Ctrl+
Nav & edit
R -> recent file
G -> goto line
T -> goto symbol in the whole workspace
H -> find & replace
`[ ]` -> adjust ident

K W -> close all tabs
(custom ) K K -> close all other tabs

` -> toggle terminal

O -> go to a symbol

## Alt
up/down -> move line of code
